## 如何让 OpenClaw 安装钉钉 AI 表格 Skill？

首先，安装一下 ClawHub 这个 skill，这是 OpenClaw 用于安装 skills 的管理器

```bash
npm i -g clawhub
```
或者直接要求你的 OpenClaw 安装 clawhub 这个 skill。

安装完后，就可以直接对你的 OpenClaw 说：
```bash
安装 dingtalk-ai-table 这个 skill
```
或者，在终端里手动安装：
```bash
clawhub install dingtalk-ai-table
```

有任何问题，请在钉钉AI表格官方交流群提问。

https://table.dingtalk.com
